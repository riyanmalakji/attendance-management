# SVIP - Attendance Management App

SVIP is an Android-based Attendance Management and Student Resource application developed as a diploma final-year project.  
The app helps students track attendance, manage academic tasks, and access important study resources in one place.

ABOUT THE PROJECT

Manual attendance tracking can be time-consuming and prone to mistakes.  
SVIP provides a simple mobile solution where students can mark attendance, view records, download lab manuals, and manage daily academic activities.

The application is built using Android Studio with Java and Firebase for authentication and database storage.

FEATURES

- User login using Email or Google authentication  
- Attendance marking and record tracking  
- Lab manual and academic resource downloads  
- To-do list for managing tasks and assignments  
- Study tips and useful student information  
- Contact form for queries or support  
- Simple and user-friendly interface  

TECH STACK

Language: Java  
IDE: Android Studio  
Database: Firebase Realtime Database  
Authentication: Firebase Authentication and Google Sign-In  
Architecture: Client-Server approach
